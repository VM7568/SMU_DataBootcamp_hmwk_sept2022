Summary: Latitude vs X

Whether financial, political, or social -- data's true power lies in its ability to answer questions definitively. So let's take what you've learned about Python requests, APIs, and JSON traversals to answer a fundamental question: "What's the weather like as we approach the equator?" The purpose of this assignment was to analyze how weather changes as you get closer to the equator. This analysis was done for a previous assignment in November of 2022, using data from OpenWeatherMap API to assemble a dataset on over 500 randomly selected cities.

After getting the data, Matplotlib was used to make scatter plots about various aspects of the weather versus Latitude. The weather aspects required were temperature, humidity, cloudiness, and wind speed.

This site provides the visualizations and data regarding this assignment analysis from November 2022. It also is, in itself, an assignment regarding the Web, HTML and learning how to create a site. As one would expect, the analysis showed that the temperature was higher the closer the city was to the equator.
